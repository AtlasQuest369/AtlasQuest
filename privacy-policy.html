Politique de Confidentialité

AtlasQuest — Apprendre le monde en jouant

📅 Dernière mise à jour : Juin 2026 — Version 1.0

1. Présentation

AtlasQuest est une application mobile éducative de géographie développée et publiée par AtlasQuest. Cette politique explique comment nous traitons vos données lorsque vous utilisez notre application.

Nous ne vendons jamais vos données personnelles à des tiers. Votre vie privée est notre priorité.

2. Données collectées

AtlasQuest collecte uniquement les données nécessaires au fonctionnement de l'application :

  • Données de progression : scores, niveaux, badges, streak (stockés localement sur votre appareil)
  • Préférences : langue choisie, statut premium (stockés localement)
  • Données publicitaires : Google AdMob peut collecter un identifiant publicitaire anonyme pour afficher des publicités pertinentes
  • Données d'utilisation : statistiques anonymes de crash et performance via Firebase

3. Stockage des données

Toutes vos données de jeu (progression, gems, vies, streak) sont stockées uniquement sur votre appareil. Elles ne sont pas transmises à nos serveurs. Si vous désinstallez l'application, ces données sont supprimées.

4. Publicités (Google AdMob)

La version gratuite d'AtlasQuest affiche des publicités fournies par Google AdMob. Ces publicités peuvent utiliser votre identifiant publicitaire pour personnaliser les annonces. Vous pouvez désactiver la personnalisation dans les paramètres de votre appareil Android sous Paramètres → Google → Annonces.

Les utilisateurs Premium (abonnés) ne voient aucune publicité.

5. Application éducative — Protection des mineurs

AtlasQuest est destinée à un public familial incluant les enfants. En conformité avec les lois sur la protection de l'enfance :

  • Nous ne collectons pas sciemment de données personnelles d'enfants de moins de 13 ans
  • Aucun chat, réseau social ou partage de données avec d'autres utilisateurs n'est présent
  • Le contenu est 100% éducatif et adapté à tous les âges
  • Les publicités sont filtrées pour être adaptées aux enfants

6. Partage des données

Nous ne partageons vos données qu'avec :

  • Google AdMob — pour afficher des publicités (version gratuite uniquement)
  • Google Play — pour la gestion des achats in-app et abonnements

Aucun autre partage n'est effectué. Nous ne vendons pas vos données.

7. Sécurité

Vos données sont protégées par les mécanismes de sécurité standard d'Android. Les communications réseau utilisent exclusivement le protocole HTTPS chiffré.

8. Vos droits

Vous avez le droit de :

  • Accéder à vos données (stockées sur votre appareil)
  • Supprimer vos données (en désinstallant l'application)
  • Refuser la personnalisation publicitaire
  • Contacter notre support pour toute question

9. Modifications

Nous nous réservons le droit de modifier cette politique. Toute modification sera notifiée dans l'application. La date de mise à jour est indiquée en haut de cette page.
